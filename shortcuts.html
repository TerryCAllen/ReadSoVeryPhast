Troubleshooting

Shortcut not appearing in Share Sheet

Make sure you've completed the installation by tapping "Add Shortcut" in the Shortcuts app. You may need to scroll down in the Share Sheet to find it the first time.

"Shortcut cannot be opened" error

Make sure you're opening the install link on your iPhone or iPad, not on a desktop computer. The shortcut only works on iOS devices.

No text extracted from page

Some pages (like PDFs or heavily JavaScript-dependent sites) may not extract well. Try copying the text manually and pasting it into the Read So Very Phast app directly.

Shortcut takes a long time

The shortcut needs to download the webpage and process it. For long articles or slow connections, this may take 5-10 seconds. Be patient!

Privacy permissions prompt

The first time you run the shortcut, iOS may ask for permission to access web pages. This is normal - tap "Allow" to let the shortcut work. Your data stays private on your device.

What the Shortcut Does

For transparency, here's exactly what the shortcut does (you can review this yourself in the Shortcuts app):

  1. Gets the URL of the current Safari page
  2. Downloads the HTML content of that page
  3. Loads the Mozilla Readability library
  4. Extracts the main article text (removing ads, navigation, etc.)
  5. Encodes the text for safe URL transmission
  6. Opens Read So Very Phast with the text pre-loaded
üîí Your Privacy: All of this happens entirely on your device. Nothing is sent to external servers. The shortcut simply passes the extracted text directly to the Read So Very Phast web app.

Alternative: Manual Copy-Paste

If the shortcut doesn't work for a particular page, you can always:

  1. Select and copy the article text in Safari
  2. Navigate to Read So Very Phast
  3. Paste the text into the input area
  4. Start reading!
